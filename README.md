## ИСПОЛЬЗОВАТЬ СТРОГО С АНДРОИД 15!!!
##  Ключевые возможности

*   **Живая карта города:** Интеграция с Yandex MapKit, отображающая базу из более чем 300 реально существующих заведений Ростова-на-Дону с точными координатами и адресами.
*   **Детальный анализ доступности:** Каждое место содержит чек-лист из 7 параметров (наличие пандуса, кнопок вызова, звуковых сигналов, тактильной плитки, широких входов, парковки для МГН и лифтов).
*   **Умная навигация:** Система построения пешеходных маршрутов, которая адаптируется под выбранную в профиле категорию (например, построение пути без лестниц и высоких бордюров для пользователей на колясках).
*   **Социальный мониторинг (UGC):** Пользователи могут ставить собственные желтые метки на карте, отмечая временные препятствия или новые доступные точки, сопровождая их описанием и реальными фотографиями.
*   **Система отзывов:** Интеграция реальных отзывов из внешних источников (Яндекс Карты) и возможность добавления пользовательских комментариев о качестве доступной среды.
*   **Синхронизация с данными города:** Функция импорта объектов из государственных реестров доступности с автоматической проверкой дубликатов и Push-уведомлениями о новых точках.

##  Технологический стек

*   **Язык программирования:** Kotlin
*   **Интерфейс:** Jetpack Compose (современный декларативный UI на базе Material Design 3)
*   **Геолокация и Карты:** Yandex MapKit 4.5.1 Full SDK
*   **Архитектура:** MVVM (Model-View-ViewModel) + State Management (сохранение состояния вкладок при навигации)
*   **Работа с медиа:** Библиотека Coil для высокопроизводительной загрузки изображений
*   **Уведомления:** Внедрена поддержка Notification Channels для Android 13+

##  Структура проекта

*   `navigation/` — конфигурация графа навигации и описание логики переходов.
*   `presentation/map.kt` — ядро системы: управление картой, GPS-позиционированием, база данных локаций и логика создания меток.
*   `presentation/search.kt` — поисковый модуль с фильтрацией по типам заведений (Кафе, Парки, Магазины и т.д.).
*   `presentation/profile.kt` — личный кабинет пользователя, настройка параметров доступности и модуль синхронизации.
*   `presentation/main.kt` — навигационная оболочка приложения с нижней панелью управления.
*   `presentation/routes.kt` — журнал пользовательской активности и сохраненных отчетов.
